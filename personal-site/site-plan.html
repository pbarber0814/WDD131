Overview

Purpose

This website is designed to share my interest in video games and highlight some of my favorite games. It provides information, images, and personal opinions about different games I enjoy. The goal is to create an engaging and visually appealing site that showcases my hobby.

Audience

The intended audience for this website is other gamers and people who are interested in video games. It is also for friends, classmates, and anyone who wants to learn more about popular games and why they are enjoyable.

Dynamic elements

JavaScript will be used to add interactive and dynamic elements to the site. I will create an array of game objects that store information such as title, image, and description. Using JavaScript functions, I will dynamically display this content on the page. The site will include event listeners, such as button clicks, to allow users to interact with the content, like showing more details about a game. I will use DOM manipulation to select elements and update the page content. Conditional statements will be used to control what content is displayed based on user interaction. Multiple functions will be used to organize the code and improve readability.

Content

Home page

Welcome to my gaming website! This site is all about my favorite video games and why I enjoy them. Gaming has been one of my favorite hobbies for years, and I enjoy exploring new worlds, completing challenges, and playing with friends. On this site, you will find information about my favorite games, images, and more.

About Page

My name is Preston, and I have enjoyed playing video games for many years. Gaming is one of my favorite hobbies because it allows me to relax, explore new worlds, and challenge myself. I enjoy a variety of games, from open-world adventures to competitive multiplayer games. This website is a way for me to share my interest in gaming with others.

Games Page

Some of my favorite games include Minecraft, Skyrim, and Assassins Creed: Valhalla. I enjoy these games because they offer different experiences such as creativity, competition, and teamwork. Each game provides unique challenges and gameplay styles that keep me engaged. Minecraft is a creative sandbox game where players can build and explore endless worlds. Skyrim is an open-world RPG that allows players to complete quests, fight enemies, and explore a large fantasy world. Assassin’s Creed Valhalla is an action-adventure game set in the Viking age, where players experience combat, exploration, and storytelling.

Gallery Page

The gallery page will include images from my favorite games. These images may include screenshots, game covers, and gameplay moments. The purpose of this page is to visually showcase the games and make the site more engaging.

Contact Page

The contact page will include a simple form where users can enter their name, email, and a message. This allows visitors to share their thoughts about the site or suggest other games. The form will be simple and easy to use.

Wireframes

Home

The home page will include a header with the site title and logo, followed by a navigation bar. The main section will feature a welcome message and introduction to the site. Below that will be a featured games section displaying a few favorite games with images and short descriptions. The page will end with a footer.

About Page

The about page will include the same header, navigation bar, and footer as the other pages for consistency. The main section will feature a heading and an image, followed by text that introduces who I am and explains my interest in video games. The layout will place the image next to the text to create a balanced and visually appealing design.

Games Page

The games page will display a list of favorite games. Each section will include an image of the game along with a description explaining why it is enjoyable. The layout will be structured in rows, making it easy to read and visually organized. The page will also include the same header, navigation bar, and footer as the home page.

Gallery Page

The gallery page will include a header and navigation bar at the top, followed by a grid layout displaying images from different video games. Each image will represent a screenshot or game cover. The images will be organized in rows and columns to create a clean and structured appearance. A footer will be included at the bottom of the page

Contact Page

The contact page will include the standard header and navigation bar at the top. The main section will contain a contact form with fields for name, email, and message, along with a submit button. The layout will be simple and centered to make it easy for users to fill out the form. A footer will be included at the bottom of the page.
